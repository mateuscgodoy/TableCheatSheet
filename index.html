HTML Tables Cheat Sheet

A simple and quick cheat sheet for HTML tables, covering all the major features from this important tag.

Code Layout

Raw HTML structure.

<table>
<thead >
<!-- Row 1 -->
<tr>
<th colspan="3">Column Span of 3 on Table Head</th>
</tr>
<!-- Row 2 -->
<tr>
<th></th>
<th>Head Header 1</th>
<th>Head Header 2</th>
</tr>
</thead>
<tbody>
<!-- Row 3 -->
<tr>
<th>Body Header 1</th>
<td>Data 1</td>
<td rowspan="2">Row Span of 2</td>
</tr>
<!-- Row 4 -->
<tr>
<th> Body Header 2</th>
<td>Data 1</td>
</tr>
<!-- Row 5 -->
<tr>
<th>Body Header 3</th>
<td>Data 1</td>
<td>Data 2</td>
</tr>
</tbody>
<tfoot>
<!-- Row 6 -->
<tr>
<th>Footer Header 1</th>
<td colspan="2">Footer Column Span of 2</td>
</tr>
<tfoot>
</table>
